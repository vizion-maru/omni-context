# Task 4.9: Keyboard Shortcuts Enhancement

Implement expanded keyboard shortcuts and slash commands in `extension/sidepanel.js`.

## Current state
The `setupKeyboardShortcuts()` function already handles: Ctrl+K (focus input), Ctrl+Shift+M (mindmap), Ctrl+Shift+F (tab search), Escape (cancel streaming).

## Add these new shortcuts

1. **Ctrl+Shift+N** — New conversation (call the existing clearConversation or newChat function - find the New Chat button click handler and trigger it)
2. **Ctrl+Shift+E** — Export conversation (trigger the existing export functionality)
3. **Arrow Up in empty input** — Recall the last user question (find last user message in `messages[]` array and put it in `inputEl.value`, then call `autoResizeInput()`)
4. **Slash commands** — When user types a slash command in the input and presses Enter, intercept before sending to AI:
   - `/search <query>` → trigger the tab search (focus `tabSearchInput`, set its value, dispatch input event)
   - `/compare` → enable compare mode (click the compare button if it exists)
   - `/summarize all` → auto-send "Summarize the content of all my indexed tabs" as the prompt

## Implementation details
- Add new shortcuts inside `setupKeyboardShortcuts()` for Ctrl+Shift+N and Ctrl+Shift+E
- For arrow-up recall: add a keydown listener on `inputEl` that checks `e.key==='ArrowUp' && inputEl.value.trim()==='' && !e.ctrlKey`
- For slash commands: intercept in the `send()` function or input keydown before calling `send()`
- Add i18n keys to `extension/_locales/en/messages.json` and `extension/_locales/de/messages.json` for any new UI strings
- Update any keyboard shortcut help/tooltip if one exists
