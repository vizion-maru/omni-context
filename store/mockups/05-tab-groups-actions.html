The torch package contains data structures for multi-dimensional tensors and defines mathematical operations over these tensors. Additionally, it provides many utilities for efficient serialization of tensors and arbitrary types, and other useful utilities.

import torch

x = torch.randn(3, 4)
y = torch.randn(4, 5)
z = torch.matmul(x, y) # Matrix multiply
print(z.shape) # torch.Size([3, 5])

It has a CUDA counterpart that enables you to run your tensor computations on an NVIDIA GPU with compute capability ≥ 3.0. The package mirrors the CPU tensor types with CUDA equivalents.
