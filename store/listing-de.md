# Omni-Context — Chrome Web Store Listing (Deutsch)

## Kurzbeschreibung (max. 132 Zeichen)

Stelle Fragen an alle offenen Tabs mit deinem eigenen AI-Key. 10 Provider, 100% privat, kein Backend. BYOK.

## Ausführliche Beschreibung

Omni-Context verwandelt deine offenen Browser-Tabs in eine durchsuchbare Wissensbasis. Stelle eine Frage — die Erweiterung liest den Inhalt deiner offenen Tabs, bewertet die Relevanz und liefert eine KI-gestützte Antwort mit Quellenangaben.

DEIN SCHLÜSSEL, DEINE KONTROLLE (BYOK)
Omni-Context speichert keinen einzigen Datenpunkt auf einem Server. Dein API-Key wird lokal in deinem Browser abgelegt. Alle Anfragen gehen direkt von deinem Browser an den KI-Anbieter deiner Wahl. Kein Backend. Kein Tracking. Keine Registrierung.

10 KI-ANBIETER — DU WÄHLST
OpenAI (GPT-4o, o3, o4) · Anthropic (Claude Sonnet 4, Opus) · Google Gemini · Groq (Llama 3.3) · Mistral · DeepSeek · xAI (Grok) · OpenRouter · Perplexity · Cohere. Modelle werden live von der jeweiligen API geladen — du hast immer Zugriff auf die neuesten Modelle.

SIDE PANEL — IMMER GRIFFBEREIT
Omni-Context lebt im Chrome Side Panel. Kein Tab-Wechsel nötig. Stelle Fragen, während du recherchierst, und behalte deine Tabs im Blick.

WAS OMNI-CONTEXT KANN

• Tab-Kontext-Chat — Stelle Fragen über den Inhalt aller offenen Tabs gleichzeitig. Omni-Context extrahiert Text aus jeder Seite (inkl. PDFs), bewertet die Relevanz per TF-IDF und baut den Kontext automatisch zusammen.

• Quellenangaben — Jede Antwort referenziert die genutzten Tabs als klickbare Chips. Ein Klick springt direkt zum Tab und hebt die relevante Passage hervor.

• Recherche-Modus — Aktiviere den Recherche-Modus für systematische Tab-für-Tab-Analyse. Ideal für Literaturrecherche, Marktanalyse oder Vergleiche.

• Relevanz-Scoring — Vor jeder Antwort siehst du, welche Tabs verwendet wurden und wie relevant sie waren (Prozent-Score pro Tab).

• Tab-Gruppen-Integration — Chrome Tab Groups werden erkannt und im Kontext-Panel angezeigt. Frage gezielt nach einer Tab-Gruppe ("Fasse meine Recherche-Gruppe zusammen").

• Kohärenz-Score — Ein Pill-Badge zeigt, wie thematisch zusammenhängend deine offenen Tabs sind. Erkenne Ausreißer sofort.

• Follow-up-Vorschläge — Nach jeder Antwort generiert Omni-Context drei kontextbezogene Folgefragen als klickbare Chips.

• Chat-Verlauf — Alle Gespräche werden lokal gespeichert. Durchsuche, lösche oder exportiere sie als Markdown.

• Markdown-Export — Exportiere jedes Gespräch als formatierte Markdown-Datei inkl. Quellenangaben.

• Mermaid-Diagramme — KI-generierte Diagramme werden direkt im Chat gerendert. Klicke auf Knoten, um Folgefragen zu stellen.

• Auto-Indexierung — Tabs werden automatisch beim Laden indexiert und alle 60 Sekunden aktualisiert. PDF-Dateien werden direkt im Service Worker extrahiert.

• Quellen-Aktionsmenü — Rechtsklick auf einen Quellen-Chip: Tab öffnen, vertiefen, vergleichen oder fehlende Infos identifizieren.

FÜR WEN IST OMNI-CONTEXT?

• Researcher und Wissenschaftler — Dutzende Paper-Tabs offen? Stelle übergreifende Fragen, ohne zwischen Tabs zu wechseln.

• Studenten — Vorlesungsunterlagen, Wikipedia, Paper — alles gleichzeitig durchsuchen und zusammenfassen lassen.

• Power-User und Wissensarbeiter — Marktanalyse, Wettbewerbsvergleich, Dokumentationsrecherche über viele Quellen hinweg.

• Entwickler — Stack Overflow, GitHub Issues, Docs — frag über alle offenen Tabs auf einmal.

DATENSCHUTZ
Kein Backend. Kein Server. Keine Datenerfassung. Kein Analytics. Kein Tracking. Dein API-Key und deine Tab-Inhalte verlassen niemals deinen Browser (außer direkt an den gewählten KI-Anbieter). Siehe unsere Datenschutzerklärung für Details.
